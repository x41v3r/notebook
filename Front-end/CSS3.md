# 一、CSS 如何使用

## 1.语法规范

![CSS语法规范](./images/CSS语法规范.png)

```css
/*
    为所有的 h1 标签设置
    文字颜色为 红色
    字体大小为 25 像素
*/
h1 {
    color: red;
    font-size: 25px;
}
```

* 选择器选择页面中要设定样式的元素，在后面紧跟的花括号内设定具体的样式（属性）
* 属性和属性值以 “键值对” 的形式书写，**每一组** “键值对” 后面都以英文分号 `;` 结尾
* 空格规范
    * 在属性值的前面，冒号的后面，保留一个空格
    * 在选择器的后面，花括号的前面，保留一个空格

# 二、CSS 选择器

## 1.基础选择器

> 标签选择器

&emsp;&emsp;用 HTML 标签名称作为选择器，按标签名分类，为页面中某一类标签**全部**指定统一的样式。

```css
标签名 {
    属性1: 属性值1;
    属性2: 属性值2;
    属性3: 属性值3;
    属性4: 属性值4;
    ...
}
```

> 类选择器

&emsp;&emsp;为多个元素设置相同的 class 属性，使用类选择器同时选中页面中所有具有相同 class 属性的元素。在开发中最为常用。 

```css
.类名 {
    属性1: 属性值1;
    属性2: 属性值2;
    属性3: 属性值3;
    属性4: 属性值4;
    ...
}
```

* 多个不同的 html 标签可以设置相同的 class 属性，设置统一的样式
* 一个 html 标签可以同时设置多个 class 属性，多个样式同时发挥作用

```html
<html>
    <head>
        <style>
            .red {
                color: red;
            }
            .bold {
                
            }
            .green {
                color: green;
            }
        </style>
    </head>
    <body>
        <ul>
            <li class="red bold">刘德华</li>
            <li class="red">薛之谦</li>
            <li class="red">毛不易</li>
            <li class="red bold">周杰伦</li>
            <li class="red bold">陈奕迅</li>
            <li class="green">理塘丁真</li>
            <li class="green">知识雪豹</li>
            <li class="green">阿力木</li>
        </ul>
    </body>
</html>
```
