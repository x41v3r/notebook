> 标识符的命名

* 标识符命名规则
    * 只能由 26 个英文字母大小写、0-9、\_ 或 $ 组成，且不能以数字开头        
    * 不能是关键字或保留字，但是可以包含关键字或保留字
    * Java 严格区分大小写，不限制标识符的长度，但是标识符内不能包含空格
* 标识符命名规范（推荐但不强制）
    * 包名：多单词组成时所字母都小写
    * 类名、接口名：多单词组成时每个单词的首字符大写
    * 变量（属性）名、方法名：多单词组成时，除第一个单词之外每一个单词的首字母都大写
    * 常量名：每一个字母都大写，多单词时每个单词之间用 `_` 连接
