> 注释

```python
"""
    打印 hello, world
    （多行注释）
"""
print("hello, world")
# 打印 hello, world （单行注释）
```

* 对于单行注释，Python 规范要求 `#` 和 `注释内容` 之间以一个空格隔开
* 多行注释一般用于解释整个 Python 代码文件、类或方法
