\subsection*{1.3 The basic structure of \LaTeX\space source file}

\begin{lstlisting}
% introduction field: set global configure
\documentclass{article} % import a document class (article book report letter)

\title{My First Document} %set the title of the document
\author{Xaiver Yang} %input the author's name
\date{\today} %time of the last edit

% text field: the content of the document
\begin{document}
	
    \maketitle  %output the title, the author and the edit time at this position
	
    Hello,\LaTeX

	$f(x)=x^2$ is a funcation. % single "$" means a inline mathematical formula
	$$f(x)=3x^2+x-1$$ % double "$" means an outline mathematical formula
\end{document}
\end{lstlisting}

\begin{itemize}
    \item a document can only contains a "document" environment
    \item we are supposed to add empty lines properly to make the format of the document cleaier
    \item inside the "text field", contents warpped by "\$", we call it "mathematical mode"; the others, we call it "text mode"
    \item adding an empty line between two "text field" lines which need to warp can realize the expected effect, we are not allowed to write anything into the empty line and a comment won't be regarded as a empty line
\end{itemize}
