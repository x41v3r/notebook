> 标识符的命名

* 命名规则
    * 只能由 26 个英文字母大小写、0-9、\_ 或 $ 组成，且不能以数字开头        
    * 不能是关键字或保留字，但是可以包含关键字或保留字
    * Java 严格区分大小写，不限制标识符的长度，但是标识符内不能包含空格
* 命名规范（推荐但不强制）
    * 包名：多单词组成时所字母都小写
    * 类名、接口名：多单词组成时每个单词的首字母都大写
    * 变量（属性）名、方法名：多单词组成时，除第一个单词之外每一个单词的首字母都大写
    * 常量名：每一个字母都大写，多单词时每个单词之间用 `_` 连接

> 变量

* Java中的变量
    * 变量必须在声明之后使用，因为只有声明后，变量才会在内存中被加载
    * 局部变量声明之后不会被自动初始化，因此局部变量在声明之后、使用之前必须先进行显式初始化
        * 声明之后未初始化的局部变量，只要我们不使用它，运行就不会报错
    * 变量都是定义在其作用域内的，各自都只在自己的作用域内发挥作用
        * 一个变量的作用域就是这个变量的声明语句所在的大括号内的范围
        * 同一个的作用域内不可以声明两个同名的变量（无论类型是否相同）
    * 变量的分类
        * 局部变量：声明在方法的形参列表或方法体内的变量
        * 成员变量（属性）：声明在类的 {&emsp;} 的内部的变量
