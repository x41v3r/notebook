# 1 寻找两个正序数组的中位数

&emsp;&emsp;给定两个大小分别为 `m` 和 `n` 的正序（从小到大）数组 `nums1` 和 `nums2`。请你找出并返回这两个正序数组的 **中位数** 。

&emsp;&emsp;算法的时间复杂度应该为 `O(log (m+n))` 。

> Java 解答：

```java
class Solution {
	public double findMedianSortedArrays(int[] nums1, int[] nums2) {
		int[] target = new int[nums1.length + nums2.length];
        int i = 0, j = 0, k = 0;
        while(i < nums1.length && j < nums2.length){
            if(nums1[i] < nums2[j]){
                target[k] = nums1[i];
                i++;
            }else{
                target[k] = nums2[j];
                j++;
            }
            k++;
        }

        //将为遍历完的那个数组的剩余内容全部放在target数组尾部
        while(i < nums1.length){
            target[k] = nums1[i];
            i++;
            k++;
        }
        while(j < nums2.length){
            target[k] = nums2[j];
            j++;
            k++;
        }

        double result;
        if(target.length % 2 == 0){
            result = (target[target.length / 2] + target[target.length / 2 - 1]) * 0.5;
        }else{
            result = (double) target[target.length / 2];
        }
        
        return result;
    }
}
```
